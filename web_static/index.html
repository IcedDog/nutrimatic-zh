高级设置

实时模式会定期刷新暂定结果；关闭时只显示最终结果。

在已解析语料的连续汉字片段中匹配模式，常见结果优先显示。

语法

  • 汉字：字面匹配
  • .：任意一个汉字
  • [] () | ? * + {m,n}：与正则表达式相同
  • 表达式&表达式：两边必须同时匹配完整结果
  • <内容>：内容可以乱序出现
  • @p(...)：拼音；@b(...)：部件；@h(...)：笔画
  • @t(...)：相同汉字；@z(...)：在内置词表中补字

示例

  • .*春.*&.{4}：含“春”的四字片段
  • <天地玄黄>：四个字任意次序
  • @t(A)@t(A)@t(B)@t(B)：AABB 结构
  • @p(shan3)水：首字读 shan 三声
  • @b(氵).{2}：三字且首字含三点水
  • @z(.士林)：补全能与“士林”成词的字
